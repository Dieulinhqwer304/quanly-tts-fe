# Suggested Commands

## Development

- Install deps (yarn lock exists): `yarn install`
- Start dev server: `yarn dev`
- Build production bundle: `yarn build`
- Preview build: `yarn preview`

## Quality

- Lint (auto-fix enabled in script): `yarn lint`
- Lint explicitly: `yarn lint:fix`
- Format files: `yarn format`
- Check formatting only: `yarn format:check`

## Local mock API

- Run JSON server for local API: `npx json-server --watch db.json --port 3000`

## Useful Linux utility commands

- List files: `ls`, `ls -la`
- Change directory: `cd <path>`
- Search text: `grep -R "pattern" src`
- Find files: `find src -name "*.ts"`
- Git status/diff/log: `git status`, `git diff`, `git log --oneline -n 20`
